= Supported Payments =
Sadad supports local KNET, VISA/MASTER, and Apple Pay.

== Installation ==
To use this plugin, Your account has to be qualified to use our API, please log in to your account and find the Client Key/Secret ID under your profile page, If it's not showing active please contact [support@sadadpay.net](mailto:support@sadadpay.net) to enable it. 
After that, follow the installation steps in [this link](https://sadadpay.readme.io/reference/wordpresswoocommerce).
